Different Categoris of elements : 

    Type of elementts : 

        1) Container element 

                contains both start and end tags

            ex : <title>Garuda careers</title>

        2) Empty Elements : 

                constains only start tag

            ex: <img />

        
        3) Block Level elements : 

            Starts on a new line.

                ex :div, p 

        4) Inline Elements :
        
            Starts on the same line 

                ex : span, a
